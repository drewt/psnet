psnet - Pretty Simple Network
=============================

Copyright Â© 2013 Drew Thoreson

psnet is an overlay network which serves as middleware for unreliable
broadcast messaging.


Building
--------

    $ make


Installation
------------

    $make install


Manuals
-------

    $ man pstrackd
    $ man psrouted
    $ man psnetrc


Git Repository
--------------

https://github.com/drewt/psnet

    $ git clone https://github.com/drewt/psnet.git
